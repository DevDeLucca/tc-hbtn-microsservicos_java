O que é Spring Boot Framework?
O Spring Boot é um framework Open Source que nasceu a partir do Spring framework
e veio para facilitar as configurações iniciais de um projeto  com relação, a aplicações, microserviços e stand alone.

Quais são os benefícios e recursos que Spring Boot fornece?
- Fácil configuração
- servidor embarcado
- gerenciador de dependencias.
- Otimização do tempo
- Aumento em produtividade

Como é feita a adição de novas dependências (JPA, Hibernate, etc…) ?
Através do arquivo pom.xml

O que é Spring Initializer?
Spring Initializr é uma ferramenta baseada na Web que gera a estrutura do projeto Spring Boot
